Drivel TV

Drivel TV is a web app that allows users to browse and create live broadcasts with their favourite video content. I built and deployed the back and front end servers to keep track of live-video timestamps, store group-chat messages, implement Youtube API and render broadcasts for users.

Tech-stack: Node.js, Express, MongoDB, Socket.io, Heroku, React, YouTube API
